# Moka

Moka is a personal project for a simple and modern PHP framework.

## Getting started

### Install your Moka clone
```bash
git clone https://github.com/AlfioEmanueleFresta/moka.git
cd moka/
sh scripts/install-deps.sh
sh scripts/configure.sh
```
